# AI 智能文件审查系统

## 项目简介
AI 智能文件审查系统是一个基于人工智能技术的自动化文档审查平台，旨在通过AI模型对各类文件进行智能分析和问题检测。系统支持多种审查场景和规则配置，能够有效提高文件审查效率和准确性。

## 核心功能

### 1. 文件审查
- 支持上传各类文档进行AI智能审查
- 实时显示审查进度和状态
- 提供详细的审查结果报告
- 支持查看原始文本和改进建议

### 2. 审查规则管理
- 可创建和管理多个审查规则
- 支持规则库的导入和导出
- 可为不同场景分配不同的审查规则

### 3. 审查场景配置
- 可创建多种审查场景（如合同审查、技术文档审查等）
- 每个场景可关联多个审查规则
- 支持场景的增删改查操作

### 4. AI模型管理
- 支持配置多种AI模型（OpenAI、Anthropic、百度文心等）
- 可设置模型参数和访问密钥
- 支持自定义AI模型接入

### 5. 用户权限管理
- 支持多角色权限控制（主管、管理员、普通用户）
- 主管可管理所有用户和规则
- 管理员可管理审查场景和模型配置
- 普通用户可提交文件审查任务

### 6. 工作台与仪表盘
- 提供直观的任务统计面板
- 显示审查总数、完成数、进行中数量
- 支持任务列表的筛选和分页
- 提供快速入口卡片，方便跳转到主要功能模块

## 技术架构
- 前端：React + TypeScript + Ant Design
- 后端：Spring Boot + MyBatis-Plus
- 数据库：PostgreSQL
- 部署：Docker + Docker Compose

## 使用方法
1. 登录系统
2. 在工作台点击"新建审查"按钮
3. 上传需要审查的文件
4. 选择合适的审查场景和AI模型
5. 提交审查任务并等待结果
6. 在审查结果页面查看详细的问题发现和改进建议

## 系统部署
本系统采用容器化部署方式，具体步骤如下：
1. 确保已安装Docker和Docker Compose
2. 克隆项目代码仓库
3. 执行 `docker-compose up` 启动所有服务
4. 访问 http://localhost:3000 进入系统界面
