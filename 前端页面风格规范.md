# 前端页面风格规范

> 本文档总结当前 AI 文件审查系统前端的视觉风格：技术基座、字体、颜色、图标、圆角/间距/阴影、各页面与组件的样式约定，便于二次开发时保持一致。
>
> 数据来源（前端实际代码）：
> - `frontend/index.html`、`frontend/src/main.tsx`、`frontend/src/App.tsx`
> - `frontend/src/styles/global.css`、`frontend/src/styles/reviewWorkspace.css`
> - `frontend/src/components/AppLayout.tsx`
> - `frontend/src/pages/*`（DashboardPage / LoginPage / ReviewWorkspacePage 等）
> - `frontend/src/utils/constants.ts`、`frontend/src/api/pipelineApi.ts`

---

## 一、技术基座与主题

| 项 | 取值 |
|---|---|
| UI 框架 | React + Vite + TypeScript |
| 组件库 | **Ant Design v5**（`antd`），通过 `ConfigProvider` 全局配置 |
| 图标库 | `@ant-design/icons`，统一使用 **Outlined（线性）** 风格 |
| 语言 | 中文（`locale=zh_CN`，`<html lang="zh-CN">`） |
| 文档标题 | `AI 智能文件审查系统` |
| 主题主色 `colorPrimary` | **`#1677ff`**（Ant Design 经典蓝） |
| 全局圆角 `borderRadius` | **6px** |
| 弹窗 | 默认居中（`modal.centered = true`） |

`ConfigProvider` 主题配置（`src/main.tsx`）：

```tsx
theme={{ token: { colorPrimary: '#1677ff', borderRadius: 6 } }}
```

> 大部分组件颜色继承 Ant Design 默认 Design Token；下文「语义色」与各页面自定义色用于在默认主题之上做点缀和状态区分。

## 六、圆角、间距、阴影与动效

### 6.1 圆角

| 元素 | 圆角 |
|---|---|
| 全局 token | 6px |
| 卡片 / 统计卡 / 操作卡 | 12px |
| 登录卡 | 16px；登录 Logo 16px |
| 工作台面板 / 概览块 / findings 卡 | 8px |
| 定位块 / 字段块 | 6px |
| 表格 | 8px |

### 6.2 间距

- 内容区外边距：`24px`（`Content margin: 24`）。
- 页面头部下间距：`20px`（`.page-header`）。
- 工作台栅格间距：`8~12px`。
- 卡片内常用 padding：`10~16px`；源文视图 `14px 16px`。

### 6.3 阴影

| 元素 | 阴影 |
|---|---|
| 卡片默认 | `0 1px 3px rgba(0,0,0,0.06)` |
| 顶栏 | `0 1px 4px rgba(0,0,0,0.04)` |
| 卡片 / 操作卡 hover | `0 8px 24px rgba(0,0,0,0.1)` / `0 6px 20px rgba(0,0,0,0.08)` |
| findings 卡 hover/active | `0 2px 10px rgba(15,23,42,0.08)` |
| 登录卡 | `0 20px 60px rgba(0,0,0,0.3)` + `backdrop-filter: blur(20px)` |
| 登录 Logo | `0 4px 14px rgba(79,140,255,0.4)` |

### 6.4 动效

- 登录卡入场：`fadeInUp 0.5s ease-out`（位移 + 透明度）。
- 卡片 hover 上浮：`translateY(-2px ~ -3px)`，过渡 `0.3s ease`。
- findings 卡过渡：`0.16s ease`（边框/阴影/背景）。
- 侧边栏收起：内容区 `margin-left` 过渡 `0.2s`。
- 侧边栏特意**移除**点击 wave 波纹与 focus 蓝色描边（避免「3D 蓝底」观感）。

## 七、关键页面 / 组件样式约定

### 7.1 整体布局（`AppLayout`）

- 经典 Ant **Sider + Header + Content** 三段式。
- **侧边栏**：白底 `#ffffff`，右侧 `1px solid #f0f0f0`，固定定位、可折叠（展开 200px / 收起 80px）；顶部品牌区高度 64，含 `FileTextOutlined` + 品牌名。选中项用「右侧 3px `#1677ff` 竖条 + 文字加粗」表示，不使用蓝色底块。
- **顶栏**：白底，sticky，高 64，含折叠按钮 + 用户头像（`colorPrimary` 底）+ 角色 Tag + 下拉菜单。

### 7.2 登录页（`LoginPage` / `global.css`）

- 深蓝紫渐变全屏背景 + 三个半透明白色圆形装饰（opacity 0.08）。
- 居中毛玻璃卡片（420px 宽、16px 圆角、半透明白 `rgba(255,255,255,0.95)`）。
- 渐变方形 Logo（64×64）+ 24px 主标题 + 14px 灰副标题；页脚 12px 半透明白文字。

### 7.3 仪表台 / 工作台（`DashboardPage`）

- 四张**统计卡**：12px 圆角、无边框、左侧 4px 渐变色条（蓝/绿/紫/橙），hover 上浮。
- 两个**管线入口卡**：32px 彩色图标 + 标题 + 13px 灰描述。

### 7.4 审查工作台（`ReviewWorkspacePage` / `reviewWorkspace.css`）

- 背景 `#f5f7fa`，自上而下：概览条 → 进度行 → 主栅格（结果面板 / 源文面板）→ 底部栅格（日志 / 运行信息）。
- **结果卡（finding-card）**：白底、8px 圆角、左侧 4px 色条按 check_status 着色，hover/active 变蓝边 `#1677ff` + 浅蓝底 `#f8fbff`。
- **源文视图**：等宽行高 1.8，命中处 `mark` 黄底高亮（`#ffe58f` / `#fff1b8`），起始节点橙色描边 `#fa8c16`。
- **日志窗口**：浅灰底，每行左侧 3px 蓝条，偶数行斑马纹，时间戳等宽灰字。
- 响应式：`max-width: 1100px` 时栅格降为单列。

### 7.5 卡片与表格（`global.css`）

- `.ant-card`：12px 圆角、无边框、轻阴影。
- 表格：表头 `#fafafa` 底 + `#262626` 加粗字；行 hover `#f0f5ff`；表体 8px 圆角；自定义细滚动条（`#d9d9d9`）。
